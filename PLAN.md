# StudyFlow – Implementation Plan

## Goal
Build a simple AI-powered study assistant for students.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Local storage (no auth)
- AI via API (abstracted)

## Core Features (MVP)
- [ ] Notes input (paste or upload text)
- [ ] AI summary generation
- [ ] Quiz generation (5–10 MCQs)
- [ ] Save summaries & quizzes locally
- [ ] Dashboard view

## User Flow
1. User opens app
2. Pastes notes
3. Clicks "Generate Summary"
4. Generates quiz
5. Saves session

## Non-Goals (Won’t Do Now)
- Authentication
- Payments
- Social sharing
- Mobile app

## Testing Strategy
- Manual click-through testing
- Summary generation flow
- Quiz generation flow

## Progress Tracking
- Mark sections complete only after working
